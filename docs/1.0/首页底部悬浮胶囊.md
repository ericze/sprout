初长 V1.0｜底部悬浮胶囊交互 UI 最终执行文档

状态：已冻结
适用对象：Codex / iOS 开发 / 设计联调 / QA
范围：首页底部核心操作区（Floating Action Bar）
原则：不假设，不妥协兼容，不自行扩展

⸻

1. 目标

底部悬浮胶囊是首页唯一的高频动作中枢。
它只承担一件事：

让用户在大拇指舒适区内，以最低认知负担完成最高频记录动作。

它不是导航，不承担模块切换，不承担统计入口，不承担历史入口。

⸻

2. 最终冻结决策

2.1 四个入口固定
	•	喂奶
	•	辅食
	•	尿布
	•	睡眠

2.2 文案固定
	•	喂奶
	•	辅食
	•	尿布
	•	睡眠

不得改成：
	•	奶
	•	奶量
	•	喂食
	•	如厕
	•	小睡

2.3 文字规格固定
	•	字号：12pt
	•	字重：Medium

2.4 默认前景色固定
	•	图标：Theme.textPrimary
	•	文字：Theme.textPrimary

不做透明度衰减，不做 85%。

2.5 睡眠按钮固定规则
	•	开始：单击
	•	结束：长按 1 秒
	•	激活态文字：哄睡中
	•	激活态图标：moon.zzz.fill
	•	激活态图标颜色：品牌主色
	•	激活态文字颜色：品牌主色
	•	轻触进行中睡眠按钮：不结束，只提示
	•	提示文案：长按结束
	•	提示气泡停留：1200ms
	•	轻触提示时：不加额外触感
	•	睡眠开始：Haptic.medium
	•	睡眠结束：Haptic.success

2.6 图标资源策略固定
	•	优先使用系统图标
	•	睡眠：固定使用
	•	moon.zzz
	•	moon.zzz.fill
	•	尿布：V1 用系统占位方案，不做自定义资源
	•	不允许使用：
	•	🟟
	•	安全别针
	•	paperclip
	•	archivebox
	•	办公语义图标

2.7 Dynamic Island
	•	本版实装
	•	睡眠开始时静默联动
	•	睡眠结束时同步关闭/结束对应状态

3. V1 图标落地策略

3.1 喂奶
	•	使用系统中最接近奶瓶语义的图标

	•	🟟类
	•	别针类
	•	paperclip类
	•	archivebox类
	•	垃圾桶类
	•	文件类
	•	医疗类

如果系统库中仍无合格图标，则使用最低存在感的中性占位图标，并在代码注释里明确标记为：

固定：

7. 交互定义

7.1 喂奶按钮

单击执行：
	1.	Haptic.light
	2.	轻微按压反馈
	3.	打开奶量 Bottom Sheet

禁止：
	•	不直接改数值
	•	不长按
	•	不直接保存

7.2 辅食按钮

单击执行：
	1.	Haptic.light
	2.	轻微按压反馈
	3.	打开辅食 Bottom Sheet

禁止：
	•	不相机直达拍摄页
	•	不跳转新页面
	•	不做 AI 入口心智

7.3 尿布按钮

单击执行：
	1.	Haptic.light
	2.	轻微按压反馈
	3.	打开尿布 Bottom Sheet

禁止：
	•	不在按钮层直接三选一
	•	不用一键直接记
	•	不在胶囊层展开次级菜单

7.4 睡眠按钮

情况 A：当前无进行中睡眠
单击执行：
	1.	Haptic.medium
	2.	按钮切换到激活态
	3.	睡眠 session 开始
	4.	联动 Dynamic Island
	5.	首页显示 ongoing sleep 状态链路

情况 B：当前有进行中睡眠，用户单击
执行：
	1.	不结束睡眠
	2.	按钮做极轻微抖动
	3.	显示 SleepHintBubble
	4.	文案：长按结束
	5.	1200ms 后自动消失

情况 C：当前有进行中睡眠，用户长按 1 秒
执行：
	1.	Haptic.success
	2.	结束睡眠 session
	3.	关闭/结束 Dynamic Island 对应状态
	4.	按钮恢复默认态
	5.	首页生成睡眠记录卡片
	6.	触发 Undo 机制

16. SwiftUI 实现伪代码结构

下面是建议的实现骨架，Codex 可直接按此结构落地。

import SwiftUI

struct FloatingActionBar: View {
    let onTapMilk: () -> Void
    let onTapFood: () -> Void
    let onTapDiaper: () -> Void
    let onStartSleep: () -> Void
    let onEndSleep: () -> Void

    let isSleepActive: Bool

    @State private var showSleepHint = false
    @State private var sleepButtonPressed = false

    var body: some View {
        ZStack(alignment: .topTrailing) {
            if isSleepActive && showSleepHint {
                SleepHintBubble(text: "长按结束")
                    .offset(x: 0, y: -44)
                    .transition(.opacity)
            }

            HStack {
                FloatingActionButton(
                    iconName: milkIconName,
                    label: "喂奶",
                    iconColor: Theme.textPrimary,
                    textColor: Theme.textPrimary,
                    onTap: {
                        Haptics.light()
                        onTapMilk()
                    }
                )

                FloatingActionButton(
                    iconName: foodIconName,
                    label: "辅食",
                    iconColor: Theme.textPrimary,
                    textColor: Theme.textPrimary,
                    onTap: {
                        Haptics.light()
                        onTapFood()
                    }
                )

                FloatingActionButton(
                    iconName: diaperIconName,
                    label: "尿布",
                    iconColor: Theme.textPrimary,
                    textColor: Theme.textPrimary,
                    onTap: {
                        Haptics.light()
                        onTapDiaper()
                    }
                )

                SleepActionButton(
                    isActive: isSleepActive,
                    onTapInactive: {
                        Haptics.medium()
                        onStartSleep()
                    },
                    onTapActive: {
                        triggerSleepHint()
                    },
                    onLongPressEnd: {
                        Haptics.success()
                        onEndSleep()
                    }
                )
            }
            .frame(maxWidth: .infinity)
            .frame(height: 68)
            .padding(.horizontal, 16)
            .background(
                Capsule()
                    .fill(Color.white.opacity(0.95))
                    .background(.ultraThinMaterial, in: Capsule())
            )
            .shadow(color: .black.opacity(0.04), radius: 15, x: 0, y: 8)
        }
        .frame(maxWidth: .infinity)
        .padding(.horizontal)
    }

    private func triggerSleepHint() {
        guard isSleepActive else { return }
        withAnimation(.easeOut(duration: 0.15)) {
            showSleepHint = true
        }
        DispatchQueue.main.asyncAfter(deadline: .now() + 1.2) {
            withAnimation(.easeIn(duration: 0.15)) {
                showSleepHint = false
            }
        }
    }

    private var milkIconName: String {
        "drop.circle" // TODO: replace with the closest approved milk/bottle SF Symbol
    }

    private var foodIconName: String {
        "bowl"
    }

    private var diaperIconName: String {
        "circle.hexagongrid" // TODO: replace with approved neutral system placeholder
    }
}

通用按钮

struct FloatingActionButton: View {
    let iconName: String
    let label: String
    let iconColor: Color
    let textColor: Color
    let onTap: () -> Void

    var body: some View {
        Button(action: onTap) {
            VStack(spacing: 4) {
                Image(systemName: iconName)
                    .font(.system(size: 24, weight: .regular))
                    .foregroundStyle(iconColor)

                Text(label)
                    .font(.system(size: 12, weight: .medium))
                    .foregroundStyle(textColor)
                    .lineLimit(1)
            }
            .frame(maxWidth: .infinity, maxHeight: .infinity)
            .contentShape(Rectangle())
        }
        .buttonStyle(FloatingActionPressStyle())
        .accessibilityLabel(accessibilityText)
    }

    private var accessibilityText: String {
        switch label {
        case "喂奶": return "记录喂奶"
        case "辅食": return "记录辅食"
        case "尿布": return "记录尿布"
        default: return label
        }
    }
}

睡眠按钮

struct SleepActionButton: View {
    let isActive: Bool
    let onTapInactive: () -> Void
    let onTapActive: () -> Void
    let onLongPressEnd: () -> Void

    var body: some View {
        Group {
            if isActive {
                activeButton
            } else {
                inactiveButton
            }
        }
        .frame(maxWidth: .infinity, maxHeight: .infinity)
        .contentShape(Rectangle())
        .accessibilityLabel(isActive ? "哄睡中，长按结束" : "开始睡眠记录")
        .accessibilityValue(isActive ? "进行中" : "未开始")
    }

    private var inactiveButton: some View {
        Button(action: onTapInactive) {
            VStack(spacing: 4) {
                Image(systemName: "moon.zzz")
                    .font(.system(size: 24, weight: .regular))
                    .foregroundStyle(Theme.textPrimary)

                Text("睡眠")
                    .font(.system(size: 12, weight: .medium))
                    .foregroundStyle(Theme.textPrimary)
            }
            .frame(maxWidth: .infinity, maxHeight: .infinity)
        }
        .buttonStyle(FloatingActionPressStyle())
    }

    private var activeButton: some View {
        VStack(spacing: 4) {
            Image(systemName: "moon.zzz.fill")
                .font(.system(size: 24, weight: .regular))
                .foregroundStyle(Theme.brandPrimary)

            Text("哄睡中")
                .font(.system(size: 12, weight: .medium))
                .foregroundStyle(Theme.brandPrimary)
        }
        .frame(maxWidth: .infinity, maxHeight: .infinity)
        .onTapGesture {
            onTapActive()
        }
        .onLongPressGesture(minimumDuration: 1.0) {
            onLongPressEnd()
        }
    }
}

struct FloatingActionPressStyle: ButtonStyle {
    func makeBody(configuration: Configuration) -> some View {
        configuration.label
            .scaleEffect(configuration.isPressed ? 0.97 : 1.0)
            .opacity(configuration.isPressed ? 0.82 : 1.0)
            .animation(.easeOut(duration: 0.12), value: configuration.isPressed)
    }
}

17. Dynamic Island 联动实现要求

睡眠按钮触发后，必须串到睡眠 session 逻辑，而不是只做 UI 状态切换。

开始睡眠时
	•	创建/启动 ongoing sleep session
	•	请求/更新 Dynamic Island 对应的 Live Activity
	•	首页按钮进入激活态
	•	首页 OngoingStateBar 也进入进行中展示

结束睡眠时
	•	结束 ongoing sleep session
	•	结束/关闭对应 Live Activity
	•	生成 sleep record
	•	首页按钮恢复默认态
	•	首页显示新增睡眠卡片
	•	进入 Undo 链路
